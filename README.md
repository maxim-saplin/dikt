## dikt / bottom-up dictionary

One-hand adapted, off-line dictionary with simplistic UI, dark and light themes

WordNetÂ® 3 (large lexical dictionary of English) is bundled with the app. Other dictionaries can be imported from JSON files. How do I get those files? You can use dictionaries available in various formats (DSL, StarDict, etc.) and convert them using PyGlossary free tool (available on GitHub) to JSON.
